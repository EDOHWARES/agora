# Agora: Organizers Feature - Design Specifications (Filled State)

This document provides a detailed technical breakdown of the "Organizers" feature for the Agora platform, focusing on the filled state based on issue #248.

## 1. Organizers Main Screen (Filled State)
The main screen is divided into two primary sections: "My Organizer profile" and "Subscribed Organizers".

### A. "My Organizer profile" Card
*   **Visuals**:
    *   **Header**: A vibrant, colorful abstract gradient (Orange/Yellow/Purple) as the card's top background.
    *   **Avatar**: Circular profile image centered below the header.
    *   **Info**: Name (e.g., "John Doe") and handle (e.g., "@john_doe") in bold black text.
*   **Action**: A yellow "Edit" button (rounded capsule shape) at the bottom right of the card.
*   **Status Indicators**: Small green dots or badges maybe used for online status or verification.

### B. "Subscribed Organizers" Section
*   **List**: A vertical list of dark-themed cards representing subscribed organizations.
*   **Card Content**:
    *   **Left**: Circular avatar/logo of the organization.
    *   **Middle**: Organization name (Bold, White) and a short description/location (Muted text).
    *   **Right**: A yellow "Unsubscribe" button (rounded capsule shape).

## 2. Global Styles & UI Motifs
*   **Background**: Soft cream-yellow (`#FDFCEE`).
*   **Typography**: Clean sans-serif (Bold headers, Medium labels).
*   **Color Palette**: Yellow accent (`#FFD900`), Deep Black (`#000000`), and vibrant rainbow gradients for headers.
*   **Shadows**: Subtle but consistent dark drop-shadows on all interactive cards.
*   **Footer**: Standard dark Agora footer remains consistent across all views.
